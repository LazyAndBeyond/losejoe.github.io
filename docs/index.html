The offical documentation of Discord.jr

How to install:
npm i discord.jr --save

How to create a client:

const Discord = require('discord.jr')
const Client = new Discord.Client(config.token, {Clientoptions});

Client options: [Object]


Client options include
prefix [string]
playing [string]
status [string]
url [string]

Register commands:

Client.Command(name, aliases, description, fulldescription, cooldown, permissions)

Values:
name [string]
aliases [array]
description [string]
fulldescription [string]
cooldown [int]
permissions [int]

Other Client Functions


Client.Command()
Client.getCommand()

Client events


command [(command, message, args)]
error [(error)]
warn [(warn)]
