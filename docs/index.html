Creating a new client:


const discordjr = require('discord.jr')
const client = new discordjr.Client({clientOptions})

Client functions:


.Command({cmdOptions}) - registers a command
.clearInterval(interval) - Clears an interval
.clearTimeout(timeout) - Clears a timeout
.createVoiceBroadcast() - Creates a voice broadcast
.destroy() - Logs out, terminates the connection to discord, and destroys the client
.fetchApplication(id) - Obtains the Oauth application of the bot from discord
.fetchInvite(invite) - Obtains an invite from Discord
.fetchUser(id, cache) - Obtains a user from Discord, or the user cache if its alreadt available
.fetchVoiceRegions() - Obtains the available voice regions from Discord.
.fetchWebhook(id, token) - Obtains a webhook from Discord
.generateInvite(permissions) - Generates a link that can be used to invite the bot to a guild
.login(token) - Logs the client in, establishing a websocket connection to Discord.
.setInterval(fn, delay, ...args) - Sets an interval that will be automatically cancelled if the client is destroyed.
.setTimeout(fn, delay, ...args) - Sets a timeout that will be automatically cancelled if the client is destroyed.
.sweepMessages(lifetime) - Sweeps all text-based channels' messages and removes the ones older than the max message lifetime.
.syncGuilds(guilds) - Requests a sync of guild data with Discord.
